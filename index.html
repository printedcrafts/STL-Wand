? How to Use — STL Wand Editor
Quick start guide for segmentation, selection, movement, explode, and export. Use the checkbox below if you do not want this to appear on the next launch.
Getting Started
STL Wand Editor is a browser-based tool for loading, inspecting, segmenting, and exporting 3D STL files — no installation required.
Loading a File
1
Drag & drop any .stl file onto the viewport, or click the LOAD button in the toolbar to browse.
2
The file is parsed entirely in your browser — nothing is uploaded to any server.
3
Once loaded, the model appears in the 3D viewport and the SEG panel opens on the left, the Parts panel on the right.
Supported: Both binary and ASCII STL formats. Files up to 256 MB and up to 3 million triangles.
Basic Workflow
1
Load your STL file.
2
Segment the model into parts using CONNECT or ANGLE mode.
3
Select parts or individual triangles with the Wand or Box Select tool.
4
Edit — delete, recolor, move, or re-orient geometry as needed.
5
Export each part as a separate STL file bundled into a ZIP.
Interface Overview
The toolbar runs across the top. The SEG panel (left) contains all segmentation and editing tools. The Parts panel (right) lists every segmented part. The 3D viewport fills the centre.
Unsaved work: The editor runs entirely in your browser with no auto-save. Export your parts before refreshing or closing the tab.
Segmentation
Segmentation splits your STL model into named parts. Open the Segmentation section in the SEG panel to configure and run it.
CONNECT Mode
Splits the model at vertices that are not physically shared — i.e. bodies that are geometrically disconnected. Best for models made of multiple separate objects that were exported together into one STL.
Use CONNECT when: your model has multiple separate bodies (e.g. screws in an assembly, multi-part prints).
ANGLE Mode
Splits the model at sharp edges where adjacent triangles exceed the Break Angle threshold. Use this when all parts are connected into one mesh but you want to separate faces or surfaces.
Recommended angles: 15–25° for mechanical/flat-faced parts. 30–45° for objects with moderate curvature. Lower = more splits.
Sliders
BREAK ANGLEOnly applies in ANGLE mode. Sets the sharpness threshold at which an edge becomes a split boundary.
MIN TRIANGLESAny part with fewer triangles than this value is merged into its nearest neighbour. Cleans up tiny fragments.
RE-SEGMENT
Applies the current mode and settings to re-split the model. Any deleted triangles are preserved — the segmentation reads the current state of the geometry, not the original file.
AI Advisor
Click AI ADVISOR in the toolbar to analyse your model's geometry. The advisor examines triangle counts, surface normals, edge sharpness distribution, and aspect ratios, then recommends whether to use CONNECT or ANGLE mode and suggests optimal settings. Requires an AI proxy to be configured (see Settings).
Wand & Selection Tools
✦ Wand Tool (W)
The Wand selects individual triangles or fills a region by surface normal angle — like Photoshop's magic wand but in 3D.
CLICKSelect one triangle (at 0% tolerance) or flood-fill a region matching the surface angle.
SHIFT+CLICKAdd individual triangles to the current selection without replacing it.
SHIFT+DRAGPaint triangles as you move the cursor — like a brush over the surface.
CTRL+CLICKToggle a single triangle in or out of the selection.
Wand Tolerance
0%Single triangle click only — the most precise selection mode.
1–25%Selects flat or co-planar faces. Ideal for mechanical parts.
26–60%Gently curved surfaces — selects an entire rounded face.
61–100%Entire connected mesh — one click selects everything.
Contiguous: When checked, the wand only fills triangles connected by shared edges. Uncheck to select all matching triangles anywhere on the mesh regardless of connectivity.
☐ Box Select (B)
Drag a rectangle over the viewport to select all triangle centroids that fall inside the box. Hold SHIFT while dragging to add to an existing selection.
Parts Panel Selection
Click any part row in the right panel to select that whole part. Hold SHIFT to range-select. Hold CTRL/CMD to toggle individual parts.
Triangle Color
With triangles selected (5 or more), use the Triangle Color section in the Wand panel to paint them a custom colour. Choose from 8 presets or any colour via the picker. The colour is applied as a persistent overlay — it does not change the exported geometry.
Triangle Groups
Select 5 or more triangles, then click ◈ GROUP SELECTED TRIANGLES in the Parts panel action bar. The group is saved under its parent part with a unique colour. Each group can be renamed, shown/hidden, re-selected, or deleted from the Parts list.
Editing
Delete
Select parts in the Parts panel and click DELETE to remove entire parts. Select triangles with the Wand and click DELETE TRIANGLES to surgically remove only those faces from the mesh. Deletions are reflected in re-segmentation.
Undo: Click UNDO in the toolbar (or press nothing — there is currently one undo level) to reverse the last deletion.
Re-Orient
Use the X / Y / Z sliders or quick-step buttons (−90, −45, 180, +45, +90) to rotate the model or a selection. Rotation is auto-baked when you release the slider — changes are written directly into the geometry.
WHOLE MODELApplies rotation to all parts together.
SELECTIONRotates only the currently selected parts around their shared centre.
RESETReturns the orientation preview to 0° on all axes.
Move (M)
Switch to Move mode using the MOVE button or press M.
DragMoves the model (or selected parts) freely on the X/Y ground plane. No modifier key required.
GIZMO HANDLESDrag the red/green/blue axis handles for constrained single-axis movement.
TO GROUNDSnaps the lowest vertex of the selection to Y = 0.
Explode View
The SPREAD slider fans all parts radially outward from the model centre, making it easy to use the Wand on inner surfaces without isolation.
The TRIANGLE EXPLODE sub-section lifts selected triangles away from their mesh so you can inspect or edit a specific face region. Select triangles with the Wand, click ⚡ EXPLODE SEL, then adjust the spread slider. Click RESET to merge them back.
Isolation
Select one or more parts, then click ISOLATE SEL to hide everything else. SHOW ALL restores visibility. Isolation is temporary — it does not affect exports.
View & Navigation
Orbit Controls
Right Mouse Button DragOrbit (rotate the camera around the model).
Middle Mouse Button DragPan (slide the camera left/right/up/down).
Scroll WheelZoom in/out. Zoom follows the cursor position for precision.
Orbit presets: Change the mouse button mapping to match your preferred 3D tool (Default, Fusion 360, Tinkercad, FreeCAD, Blender, Maya) in Settings → Orbit Preset.
View Cube
The cube in the bottom-left shows the current camera orientation. Click any face label (FRONT, BACK, LEFT, RIGHT, TOP, BTM) to snap to that view with a smooth animation. Drag the cube to orbit the camera directly.
Toolbar View Modes
ORBITDefault navigation mode. Left-drag to orbit.
WAND WTriangle selection mode. Click/drag to select faces.
BOX SEL BRectangle selection mode. Drag to select a region.
MOVE MTranslation mode. Drag to move parts.
HUD Badges
Floating badges appear when certain modes are active — EXPLODE ACTIVE when the spread slider is above 0, and ROTATION PREVIEW when the orient sliders are non-zero. Each badge has a ⠿ drag handle — grab it to reposition the badge anywhere in the viewport.
Grid & Axes
The grid represents the ground plane (Y = 0). The axis indicator (X = red, Y = green, Z = blue) shows world orientation. Both can be toggled in Settings → View.
Export
STL Wand Editor exports each segmented part as its own binary STL file, bundled together in a single .zip download.
Export All Parts
Click EXPORT in the toolbar or ⬇ EXPORT ALL PARTS in the SEG panel to export every part. A modal will appear to configure the output.
Export Selected Parts
Select parts in the Parts panel first, then click ⬇ EXPORT SELECTED to export only the chosen parts.
Export Options
FILE PREFIXThe base name used for each exported file — e.g. "widget" produces widget_part_001.stl, widget_part_002.stl, etc.
KEEP ORIGINALExports each part with its current world position preserved.
CENTER EACH PARTRe-centres each part to the coordinate origin before export — useful for individual printing.
Bake before exporting: If you've used Re-Orient or Move, the changes are auto-baked into the geometry as you work. Check the status bar for confirmation before exporting.
Tip: Export early and often. The browser does not auto-save your session — a refresh will lose unsaved edits.
Settings
Click the ⚙ SETTINGS button in the toolbar to open the settings panel.
SEG Panel Sections
Toggle which sections are visible in the left SEG panel. Hidden sections are removed entirely from the layout. Use this to declutter the panel by hiding tools you don't use.
Orbit Preset
Choose a mouse button mapping that matches your preferred 3D software. Options: Default, Fusion 360, Tinkercad, FreeCAD, Blender, Maya.
View
VIEW CUBEShow or hide the orientation cube in the bottom-left of the viewport.
GRIDShow or hide the ground plane grid.
INVERT ZOOMReverses the scroll wheel zoom direction.
Typography
Change the UI and monospace fonts used throughout the interface. Choose from 8 options for each. Adjust the base font size from 9px to 14px. Click ✓ APPLY FONT CHANGES to apply — fonts are saved to your browser and persist across visits.
SEG Panel Layout
The SEG panel sections can be reordered by drag-and-drop. A ⠿ handle on each section header lets you drag it to a new position. Click LOCK LAYOUT to freeze the order and prevent accidental moves. Unlock to rearrange again.
AI Advisor Proxy
The AI Advisor feature requires a backend proxy URL. Click ⚙ AI PROXY in the AI Advisor modal footer to configure the URL of your Cloudflare Worker proxy. The proxy holds your Anthropic API key server-side — no key is ever stored in your browser.
